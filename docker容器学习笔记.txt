ThinkPHP Builder.php SQL注入漏洞
获取环境:
拉取镜像到本地
$ docker pull medicean/vulapps:t_thinkphp_1
启动环境
$ docker run -d -p 80:80 medicean/vulapps:t_thinkphp_1
-p 80:80 前面的 80 代表物理机的端口，可随意指定。

使用与利用
访问 http://你的 IP 地址:端口号/, 假设启动的端口号为 80
